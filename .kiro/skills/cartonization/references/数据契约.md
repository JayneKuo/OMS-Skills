# 装箱相关数据契约

> 本文件提取自主 PRD 数据契约中与装箱引擎直接相关的字段定义。

## SKU 装箱必需字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| sku_id | string | 是 | SKU 唯一标识 |
| sku_name | string | 是 | SKU 名称 |
| weight | decimal | 是 | 单件重量（kg） |
| length | decimal | 是 | 长（cm） |
| width | decimal | 是 | 宽（cm） |
| height | decimal | 是 | 高（cm） |
| temperature_zone | enum | 是 | 温区：常温 / 冷藏 / 冷冻 |
| hazmat_type | enum | 否 | 危险品类型：无 / 易燃 / 易爆 / 腐蚀。默认无 |
| oversize_flag | boolean | 否 | 是否超大件，默认 false |
| must_ship_with | array | 否 | 必须同包的 SKU ID 列表 |
| cannot_ship_with | array | 否 | 禁止同包的 SKU ID 列表 |
| is_gift | boolean | 否 | 是否赠品，默认 false |
| fragile_flag | boolean | 否 | 是否易碎品，默认 false |

## 订单装箱相关字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| order_items | array | 是 | 订单商品列表，每项包含 sku_id、quantity |
| max_package_count | int | 否 | 最大包裹数，默认 5 |
| gift_same_package_required | boolean | 否 | 赠品是否必须同包，默认 true |

## 仓库装箱相关字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| max_package_weight | decimal | 是 | 单包最大重量限制（kg） |
| max_package_dimension | object | 是 | 单包最大尺寸限制（长/宽/高，cm） |
| packaging_cost_per_package | decimal | 是 | 单包包材成本 |

## 承运商装箱相关字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| max_weight | decimal | 是 | 单包最大重量（kg） |
| max_dimension | object | 是 | 单包最大尺寸（长/宽/高，cm） |
| dim_factor | int | 是 | 体积因子（快递 6000，物流 5000） |

## 数据缺失处理

| 缺失数据 | 策略 |
|----------|------|
| SKU 重量/尺寸缺失 | 使用品类平均值替代，标记为"数据降级" |
| SKU temperature_zone 缺失 | 按常温处理，标记为"数据降级" |
| SKU hazmat_type 缺失 | 按非危险品处理，标记为"数据降级" |
